# Nico Dann — Drum Lessons Website

A single-page Next.js site for Nico Dann Drum Lessons.

## Stack

- **Next.js 16** (App Router, Turbopack)
- **React 19**
- **Tailwind CSS 4** (CSS-first config via `@theme`)
- **TypeScript**
- **Google APIs** (googleapis)

## Deployment

Deployed on Netlify at [drumlessons.nicodann.com](https://drumlessons.nicodann.com)
